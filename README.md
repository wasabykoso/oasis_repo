# Oasic — Книга

Веб-приложение для чтения интерактивной книги в жанре фэнтези.

## Файлы

- `index.html` — Главная страница с обложками книг
- `oasis_book.html` — Читалка с полным текстом книги
- `reader.html` — Альтернативная версия читалки
- `book.html` — Краткая информация о книге

## Запуск

Просто откройте любой HTML-файл в браузере.
